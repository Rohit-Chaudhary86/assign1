Task 5: Difference between justify-items vs justify-self Property Works On Description justify-items Entire container (parent) Aligns all grid items horizontally within their cells. justify-self Individual grid item (child) Aligns a single item horizontally within its cell. Code Example:
1
2
Box 1 is centered due to justify-items: center (applied to the container). Box 2 is pushed to the right (end) due to justify-self: end (applied individually).
