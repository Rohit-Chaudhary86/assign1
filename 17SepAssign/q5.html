Top Features of HTML5

1. Semantic Tags

HTML5 introduced semantic tags like <header>, <footer>, <article> that make code more meaningful and accessible.

2. Multimedia Support

HTML5 includes native support for audio and video using <audio> and <video> without external plugins.

3. Form Enhancements

HTML5 provides new form input types like email, date, range, and built-in validation for better user experience.
